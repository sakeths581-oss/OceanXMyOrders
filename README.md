# OceanX My Orders UI

## Tech Stack
- Kotlin
- XML
- RecyclerView
- Material Design Components

## Features
- My Orders Screen
- Search Bar
- Filter & Sort Buttons
- RecyclerView Order Listing
- Invoice Button
- Book Again Button
- Bottom Navigation
- Profile Screen

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on Emulator or Android Device
